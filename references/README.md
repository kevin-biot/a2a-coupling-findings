# References

Papers and artefacts on which the framing of this work depends.

## Primary dependency

- [gagne-sentinel.md](./gagne-sentinel.md) — Jason Gagne's SENTINEL
  framework (v1 foundational, v2 primary anchor) and the
  accidental-governance thesis.

## Scope

This directory currently ships only the references that are already
load-bearing for the repo's public claims. Placeholder reference lists
and uncured reading notes should stay out until they are promoted into
actual citation-bearing documents.

## Companion surfaces

- [Blast Radius Framework](https://github.com/kevin-biot/blast-radius-framework) — attestation companion.
- [Governance Failure Patterns](https://github.com/kevin-biot/governance-failure-patterns) — diagnostic companion.
- [PACT-public](https://github.com/kevin-biot/pact-public) — ontology
  pack / SHACL-shape companion for inspecting the published pack form.
  This is the public shape reference, not the exact unpublished banking
  test-pack set used in the DOP rig.

## Selection discipline

Each reference earns its place by being load-bearing for at least one
specific claim in the work. Speculative or general-interest references
do not ship.
