## Promotion Rule

A result belongs on the primary track only if it is:

- measured
- attributed
- scope-bounded
- stated separately from architectural or standards implications

Comparator results, including DOP-side contrasts, stay secondary unless
they are strictly necessary to interpret the A2A observation.
