# Methods

This directory should support the **A2A finding**, not compete with it.

## Reading Order

- [t2-fanout-coupling.md](./t2-fanout-coupling.md) — primary method
  note for the current measured A2A result.
- [rig-adaptation.md](./rig-adaptation.md) — what was reused and what
  was adapted.
- [signal-class-description.md](./signal-class-description.md) —
  class-level signal descriptions and disclosure boundary.
- [hub-spoke-vs-mesh.md](./hub-spoke-vs-mesh.md) — related stricter
  replication line; not the primary landed measurement in this repo.

## Scope Rule

Method notes in this directory should explain:

- what configuration was studied
- what changed relative to the external anchor
- what class of signal is being discussed

They should not become the main narrative surface for repair patterns,
companion frameworks, or publication politics.
